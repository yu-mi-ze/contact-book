# 通讯录应用

这是一个基于HTML、CSS和JavaScript开发的通讯录应用，支持联系人管理、收藏、多种联系方式和导入导出功能。

## 功能特性

### 基础功能
- ✅ **联系人管理**：添加、编辑、删除联系人
- ✅ **收藏联系人**：标记重要联系人，方便快速查找
- ✅ **多种联系方式**：支持添加电话、邮箱、微信、地址等多种联系方式
- ✅ **导入导出**：支持Excel文件的导入和导出

### 用户体验
- 📱 **响应式设计**：适配各种屏幕尺寸
- 🎨 **现代化界面**：简洁美观的用户界面
- 💾 **本地存储**：使用localStorage保存数据，无需服务器

## 技术栈

- **前端框架**：纯HTML/CSS/JavaScript
- **样式**：自定义CSS
- **Excel处理**：SheetJS (xlsx.js)
- **存储**：浏览器localStorage

## 使用说明

### 运行项目

直接用浏览器打开`index.html`文件即可运行应用。

### 功能操作

1. **添加联系人**：
   - 点击"添加联系人"按钮
   - 填写姓名和至少一种联系方式
   - 可选择添加多种联系方式
   - 点击"保存"完成添加

2. **编辑联系人**：
   - 点击联系人卡片上的"编辑"按钮
   - 修改信息后点击"保存"

3. **删除联系人**：
   - 点击联系人卡片上的"删除"按钮
   - 确认后删除

4. **收藏联系人**：
   - 点击联系人卡片上的"收藏"按钮
   - 收藏的联系人会显示五角星标记并排在列表前面

5. **添加联系方式**：
   - 在添加/编辑联系人时，点击"添加联系方式"按钮
   - 选择联系方式类型并填写内容

6. **导入联系人**：
   - 点击"导入"按钮
   - 选择Excel文件（.xlsx, .xls, .csv）
   - 系统会自动解析并添加联系人

7. **导出联系人**：
   - 点击"导出"按钮
   - 系统会生成Excel文件并下载

## Excel导入格式

导入的Excel文件应包含以下列（支持中英文）：
- 姓名/Name
- 电话/Phone
- 邮箱/Email
- 微信/WeChat
- 地址/Address
- 备注/Notes
- 收藏/Favorite (是/否)

### 核心类

1. **Contact**：联系人数据模型
2. **ContactStorage**：联系人存储管理
3. **ContactApp**：应用控制器，处理用户交互

### 主要功能模块

- **数据管理**：联系人的增删改查和本地存储
- **用户界面**：联系人列表、表单、模态框等组件
- **导入导出**：Excel文件的解析和生成
